# よく噛んでゆっくりたべよう

食事のペースを整えるスマホファーストのMVPです。

## 起動方法

このディレクトリで以下を実行します。

```bash
python3 -m http.server 4173
```

ブラウザで開きます。

```text
http://localhost:4173/
```

## 実装済み

- トップ画面
- 今日の作戦選択
- 20分食事タイマー
- リマインド文言切り替え
- 5分・10分の満腹度チェック促し
- 満腹度記録
- 食後振り返り
- localStorage保存
- 記録一覧
- 連続ゆっくり日数の見える化
- 「いま食べすぎそう」画面
- 設定画面
- 音とバイブレーション設定
- 咲耶・ゴコウのキャラ画像表示
- 設定画面で咲耶・ゴコウを選択
- 選択した1キャラだけが全画面で応援
- 咲耶はやさしい女の子寄り、ゴコウは男の子寄りの文体
- 咲耶・ゴコウの静止画像表示

## 注意

咲耶・ゴコウの画像は、`assets/characters/` に配置しています。
公開・配布する場合は、素材の利用条件を確認してください。
